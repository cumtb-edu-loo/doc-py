# 数字和数学模块¶

本章介绍的模块提供与数字和数学相关的函数和数据类型。 [`numbers`](numbers.md#module-numbers "numbers: Numeric abstract base classes \(Complex, Real, Integral, etc.\).") 模块定义了数字类型的抽象层次结构。 [`math`](math.md#module-math "math: Mathematical functions \(sin\(\) etc.\).") 和 [`cmath`](cmath.md#module-cmath "cmath: Mathematical functions for complex numbers.") 模块包含浮点数和复数的各种数学函数。 [`decimal`](decimal.md#module-decimal "decimal: Implementation of the General Decimal Arithmetic  Specification.") 模块支持使用任意精度算术的十进制数的精确表示。
