# 开发工具¶

The modules described in this chapter help you write software. For example, the [`pydoc`](pydoc.md#module-pydoc "pydoc: Documentation generator and online help system.") module takes a module and generates documentation based on the module's contents. The [`doctest`](doctest.md#module-doctest "doctest: Test pieces of code within docstrings.") and [`unittest`](unittest.md#module-unittest "unittest: Unit testing framework for Python.") modules contains frameworks for writing unit tests that automatically exercise code and verify that the expected output is produced.
