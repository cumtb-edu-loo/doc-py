# Python 语言参考手册¶

本参考手册介绍了 Python 句法与“核心语义”。在力求简明扼要的同时，我们也尽量做到准确、完整。有关内置对象类型、内置函数、模块的语义在 [Python 标准库](3.标准库/index.md#library-index) 中介绍。有关本语言的非正式介绍，请参阅 [Python 教程](0.Python%20教程.md#tutorial-index) 。对于 C 或 C++ 程序员，我们还提供了两个手册：[扩展和嵌入 Python 解释器](0.扩展和嵌入%20Python%20解释器.md#extending-index) 介绍了如何编写 Python 扩展模块，[Python/C API 参考手册](10.C%20API接口/index.md#c-api-index) 则详细介绍了 C/C++ 的可用接口。
